# Bit-links

## Description

Bit-links is a modern, fast, and scalable URL shortening application built with [Next.js](https://nextjs.org). It enables users to create and manage shortened links effortlessly, providing a clean and intuitive interface backed by a powerful serverless API.

---

## Key Features

- **URL Shortening:** Easily generate short and shareable links.
- **Dynamic Routing:** Access shortened URLs through unique handles.
- **API Endpoints:** Create and retrieve links programmatically.
- **Optimized Fonts:** Utilizes Vercel’s Geist font for a polished UI.
- **Serverless Functions:** Built with Next.js API routes for seamless backend logic.
- **MongoDB Integration:** Reliable data storage managed through a custom library.
- **Responsive Design:** Fully optimized for desktop and mobile devices.

---

## Technologies Used

- [Next.js](https://nextjs.org) — React framework for production
- [React](https://reactjs.org) — UI library
- [MongoDB](https://www.mongodb.com) — NoSQL database
- Vercel Font Optimization ([Geist](https://vercel.com/font))
- PostCSS for CSS processing
- ESLint for code quality
